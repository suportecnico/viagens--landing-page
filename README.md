# viagens--landing-page
Projeto de certificação 1 – Agência de Viagem

#Landing Page - Viagens CamposTur

Uma **landing page de viagens** criada para os destinos, serviços, pontos turíticos, localização da empresa e dicas prátricas de viagem.

#Sobre o Projeto

A página é estruturada com HTML-5 e CSS-3, que mostra:

- Destinos paradisíacos e serviços personalizados;
- Seções, como: "Minha Viagem", "Onde nos Encontrar", "Conselhos Úteis";
- Dicas e boas práticas voltadas para melhorar a experiência do usuário;

#Pré-visualização

A página contém imagens de destinos como, Fernando de Noronha, Porto de Galinhas, Gramado, Nova York, Paris; Informações de contatos, como e-mail, telefone e whatsapp e dicas para um bom planejamento para os viajantes.
